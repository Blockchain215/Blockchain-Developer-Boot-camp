Raft Consensus Tutorial

Please go to the Secret Lives of Data website to learn about the basic mechanics of the Raft consensus protocol. While Raft is a production-ready consensus protocol, we are mainly using it here to show you a concrete example of a distributed system maintaining state across different nodes in a network.

Please note, Raft is not a Byzantine fault tolerant consensus protocol.
